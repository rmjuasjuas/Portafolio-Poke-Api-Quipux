# frontend-Quipux

**Quipux - Portafolio POKE - API**  
Sistema web interactivo para la visualización, búsqueda y gestión personalizada de Pokemon utilizando la **PokéAPI** 

# Porque lo hize?
Pokemon para mi representa muchas cosas en mi vida fue el primer anime que vi, me siento muy emotivo al ver o hacer algo que tenga que  er con pokemon me gusta mucho de verdad me apasiona pokemon y hacer lo que me gusta en este caso ser un progrmador e integralo con el plus de pokemon lo hago realmente con toda la alegria del mundo.

---

## Características Principales

### Módulo de Búsqueda Pokémon
- Consulta dinámica de Pokémon desde la **PokéAPI**  
- Visualización de imágenes 
- Parámetros configurables: límite de resultados y offset  
- Animaciones y diseño inspirado en Retro Pokemon 

### Módulo de Library
- Sistema de guardado de Pokemon favoritos 
- Funcionalidad para eliminar Pokémon de la colección  

---

## Tecnologías Utilizadas

### Frontend
- **Angular** - Framework principal del proyecto  
- **TypeScript** - Lenguaje de programación base  
- **HTML / CSS** - Estructura y estilo
- **PokéAPI REST** - Fuente oficial de datos

---

## Herramientas de Desarrollo
- **Visual Studio Code** - Editor principal para Angular  
- **Node.js / npm** - Entorno de ejecución y manejo de dependencias  
- **GitHub** - Documentación   

---

## Funcionalidades Destacadas
- Búsqueda de Pokémon por lotes (limit y offset)
- Añadir o eliminar Pokémon de la Library
- Diseño pixel inspirado en la PokeDex
